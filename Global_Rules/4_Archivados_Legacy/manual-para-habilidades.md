## 📘 Manual de Usuario: Ecosistema de Habilidades v.01

Este ecosistema está diseñado como una **cadena de producción**. Cada habilidad recibe el relevo de la anterior para asegurar que el resultado final sea coherente, estético y funcional.

### 🔗 El Flujo de Trabajo Maestro (Workflow)

Para proyectos desde cero, sigue este orden lógico:

#### 1. Fase de Cimentación (Prompt 1)

* **Cuándo usarlo:** Al inicio de un nuevo proyecto o cuando necesites una funcionalidad que la IA aún no tiene (ej: "Necesito una habilidad que traduzca código de Python a Rust").
* **Acción:** Define las reglas del juego para que el resto de las habilidades operen bajo un estándar técnico.

#### 2. Arquitectura y Estrategia (Prompt 3 + Prompt 4)

* **Cuándo usarlo:** Cuando ya sabes qué quieres construir.
* **Interacción:** * Lanza el **Prompt 3 (Arquitecto)**. Él activará el protocolo de 6 fases.
* En la **Fase 1**, integra el **Prompt 4 (Marketing)** para que la estructura no solo sea técnica, sino que esté diseñada para vender y captar leads.
* *Nota:* No dejes que el Arquitecto programe (Fase 4) hasta que el diseño esté validado.



#### 3. Estética y Visualización (Prompt 2)

* **Cuándo usarlo:** Durante la **Fase 3** del Arquitecto.
* **Acción:** Invoca esta habilidad para generar el "look & feel" futurista. Este prompt le da a la IA las instrucciones de CSS (Tailwind) y Layout (Next.js) para que la interfaz sea impactante.

#### 4. Iteración y Pulido (Prompt 5)

* **Cuándo usarlo:** Una vez que tienes el primer borrador pero sientes que falta información o que el formulario no es lo suficientemente claro.
* **Acción:** Aplica este prompt para reorganizar las secciones (Header -> Info -> Form) sin romper el diseño visual que ya creaste con el Prompt 2.

---

### 🛠 Consejos Pro de Ingeniería de Prompts

* **Uso de Memoria:** Si tu plataforma lo permite, asegúrate de que el **Prompt 3 (Arquitecto)** sea la "Habilidad Maestra" que coordine a las demás.
* **Control de "Alucinaciones":** En la **Fase 5 (Testing)** del Arquitecto, oblígalo a decirte explícitamente qué errores encontró. Si dice "todo está perfecto" a la primera en un código largo, pídele que "busque vulnerabilidades de seguridad en el formulario".
* **Personalización de Marcas:** En el **Prompt 2 (Diseño)**, puedes cambiar las palabras clave de color (ej: de "Púrpura Neón" a "Verde Esmeralda") para adaptar el diseño a diferentes clientes rápidamente.
